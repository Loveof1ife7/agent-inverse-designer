# P222 几何-性质配对数据集说明

生成日期：2026-06-20

## 1. 数据集内容

本数据集包含 P222 群的 100000 个结构样本，样本 ID 范围为：

```text
0 - 99999
```

每个样本由一份几何文件和一份性质文件组成：

```text
geometry/<id>.txt    <->    properties/<id>.csv
```

例如：

```text
geometry/12345.txt
properties/12345.csv
```

表示同一个结构样本。

## 4. 几何文件格式

路径：

```text
geometry/<id>.txt
```

每个 txt 描述一个 4x4x4 扩胞后的杆系结构，包含两类 Python 风格数据：

```text
node_data = [
    [node_id, x, y, z],
    ...
]

element_conn = [
    [start_node_id, end_node_id],
    ...
]
```

含义：

- `node_data`：节点坐标表。
- `element_conn`：杆连接表，每一行是一根杆的两个端点节点编号。
- 坐标单位：mm。
- 杆半径仿真口径：1 mm。

## 5. 性质文件格式

路径：

```text
properties/<id>.csv
```

每个 CSV 是对应结构的 Abaqus 压缩仿真应力-应变曲线。规范字段为：

```text
Strain,Stress
```

含义：

- `Strain`：工程应变，无量纲。
- `Stress`：工程应力，单位 MPa。

这些 CSV 是原始曲线性质数据，不是指标汇总表。

## 6. 仿真口径

本数据集对应的项目仿真设置如下：

- 软件：Abaqus/Explicit
- 模型：B31 梁单元杆系 lattice
- 扩胞：4x4x4
- 压缩方向：z 方向
- 目标压缩应变：0.30
- 接触：上下刚性板 + general contact
- 摩擦系数：0.3
- 单位制：mm-N-MPa-tonne
- 主材料口径：TPU95A
- 杨氏模量：8.925 MPa
- 泊松比：0.48
- 屈服应力：3.207 MPa

## 7. 使用方式

读取样本时，以文件名中的整数 ID 为唯一连接键：

```text
id = 12345
geometry_path   = geometry/12345.txt
property_path   = properties/12345.csv
```

不要把本数据集与普通标准流水线目录 `outputs/P222/` 混用；P222 在项目文档中明确属于早期特殊批次。
