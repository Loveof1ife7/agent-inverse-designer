# P222 Paired Geometry-Property Dataset

This package contains 100000 paired samples from the special P222 batch.

- Geometry: `geometry/<id>.txt`
- Property curve: `properties/<id>.csv`
- ID range: `0-99999`

The pairing key is the file stem:

```text
geometry/12345.txt <-> properties/12345.csv
```
