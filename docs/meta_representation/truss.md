# 三维晶体群约束杆系超材料：表示的数学定义、数据结构与仿真方法

本文给协作者说明本项目目前用的结构表示：它在数学上是什么、用什么数据结构存、以及如何送进 Abaqus 做大变形仿真。只讲定义和方法，不涉及代码实现细节。

项目目标：在三维晶体变换群约束下批量生成杆系超材料单胞，做大变形压缩仿真，进而做材料筛选。

---

## 一、表示的数学定义

### 1.1 基本域与对象系统

取归一化正方体作为基本域：

$$C = [0,1]^3$$

在 $C$ 上定义三类几何对象：

- **8 个顶点**：坐标固定为 $\{0,1\}^3$ 的 8 个组合，编号固定。顶点不带参数。
- **12 条棱**：每条棱由两个顶点连成，编号 $A_1,\dots,A_{12}$。
- **6 个面**：每个面由 4 个顶点围成，记为 front / back / left / right / top / bottom。

### 1.2 参数化（27 个自由度）

在上述对象上放置 **19 个可动点**，每个点的位置由若干参数控制：

- **12 个边点**：每条棱 $A_i$ 上放 1 个点，位置由一个参数 $k_i\in[0,1]$ 沿棱线性插值决定。设棱两端点为 $v_a,v_b$，则边点为

$$P_{A_i} = (1-k_i)\,v_a + k_i\,v_b .$$

  $k_i=0$ 时点在 $v_a$，$k_i=1$ 时点在 $v_b$。例如棱 $A_1$ 连接顶点 $(1,0,1)\to(0,0,1)$，边点为 $(1-k_{A_1},\,0,\,1)$。

- **6 个面点**：每个面上放 1 个点，被限制在该面所在平面内，由 2 个参数 $q$ 决定。例如 front 面（$y=0$）的面点为 $(p_{fx},\,0,\,p_{fz})$。6 个面共 12 个 $q$ 参数。

- **1 个内部点**：位于体内，坐标 $v_{\text{center}}=(v_x,v_y,v_z)$，3 个参数。

合计自由度：

$$\underbrace{12}_{k}+\underbrace{12}_{q}+\underbrace{3}_{v}=27 .$$

于是一个"未加约束的几何骨架"对应参数向量

$$p=(k_1,\dots,k_{12},\,q_1,\dots,q_{12},\,v_x,v_y,v_z)\in[0,1]^{27}.$$

### 1.3 群作用与对称约束（核心）

每个三维晶体变换群 $G$ 由一组 $4\times4$ 齐次矩阵 $\{M_s\}$ 和一组周期平移向量 $\{T\}$ 描述。矩阵作用在齐次坐标上：

$$
\begin{bmatrix}x'\\y'\\z'\\1\end{bmatrix}
=\begin{bmatrix}R & t\\ 0 & 1\end{bmatrix}
\begin{bmatrix}x\\y\\z\\1\end{bmatrix},
$$

其中 $R$ 是 $3\times3$ 旋转/反射矩阵，$t$ 是平移（体现滑移面、螺旋轴等）。

**对称约束的含义**：合法结构必须在群 $G$ 作用下不变。具体地，若某个群元素 $M_s$（必要时再加一个周期平移 $T$）把棱（或面）对象 $O_i$ 映射到对象 $O_j$ 上，则这两个对象上附着的可动点也必须重合：

$$M_s\,P_{O_i} + T = P_{O_j}.$$

把这个向量等式拆成坐标分量，就得到关于参数 $p$ 的**线性方程**。遍历群的所有元素和所有对象配对，收集到一个线性方程组

$$A\,p = b.$$

求解后把 27 个参数分成两类：

- **自由变量 free_vars**：可以独立取值的参数，个数 $=27-\operatorname{rank}(A)$；
- **从属变量 solved**：其余参数，每个写成自由变量的仿射函数。

例如群 Aba2 解出 10 个自由变量，从属变量形如

$$k_{A_1}=1-k_{A_7},\qquad p_{fx}=1-p_{bx}.$$

含义：边点 $A_1$ 的位置不能独立选，它由 $A_7$ 决定，因为群把这两条棱映射到了一起。**这一步是整个表示的关键**——它把"随便摆点"压缩到只在满足对称性的低维空间里取值，避免生成大量违反群约束的伪结构。

### 1.4 一个"结构"的完整定义

确定参数只给出 19 个点的坐标，还要选哪些点之间连杆。因此一个合法结构由三部分唯一确定：

$$\text{结构} \;=\; \big(\,\text{群 } G,\;\; \text{自由变量取值},\;\; \text{杆件集合 } E\,\big),$$

其中 $E$ 是 19 个点之间所选边的集合（点对的子集）。自由变量取值经 solved 映射恢复全部 27 个参数，得到 19 个点坐标；$E$ 给出连接关系。生成时自由变量在离散值集合中采样（$k\in\{0,0.25,0.5,0.75,1\}$，$q\in\{0.25,0.5,0.75\}$），$E$ 先取生成树保证连通再随机补边，并要求满足连通性、相对密度、杆间距等有效性条件。

## 二、对应的数据结构

按生成链路，结构表示落在以下几种文件里。每个群独立一套。

**1. 群矩阵库** `不同群的变换矩阵.json`（全局唯一，含 79 个群）

存每个群的对称操作，是 1.3 节里 $\{M_s\}$、$\{T\}$ 的数据形式：

```
groups.<群名>.M_sym         # 一组 4×4 齐次矩阵
groups.<群名>.T_vec_np      # 周期平移向量
groups.<群名>.lattice_lengths  # 各轴晶格长度，扩胞时用
```

**2. 约束文件** `constraints_<群名>.json`

1.3 节求解结果的数据形式，是"自由变量 → 全部参数"的映射：

```
free_vars   # 可独立采样的参数列表
solved      # 从属变量 = 自由变量的仿射函数，如 "k_A1": "1 - k_A7"
independent_equations  # 独立约束方程集（A p = b 的最简形式）
```

**3. 结构样本表** `<群名>-architecture.csv`

每群一张表，**一行就是一个结构**：记录该结构采样得到的参数值和它的杆件集合 $E$。每群 25000 行（即 25000 个结构）。

**4. 单胞几何** `abaqus_txt/<编号>.txt`

把某一行结构按群的全部对称操作拼成完整单胞、合并重合节点后的几何，物理单位（已按相对密度锁定的边长 $L$ 缩放）。格式是两张表——**节点坐标表**和**杆连接表**：

```python
# 节点表：[节点号, x, y, z]
node_data = [
    [1, 0.000000, 0.000000, 0.000000],
    [2, 0.000000, 0.000000, 28.008452],
    [3, 0.000000, 14.004226, 0.000000],
    ...
]

# 杆表：[起点节点号, 终点节点号]
element_conn = [
    [1, 3],
    [1, 4],
    [3, 4],
    ...
]
```

这就是杆系最朴素的图表示：`node_data` 给点的坐标，`element_conn` 给哪两个点之间有一根杆。

**5. 最终扩胞模型** `crystal_4x4x4/<编号>.txt`

格式与单胞完全相同（`node_data` + `element_conn`），但把单胞按 $4\times4\times4$ 阵列拼接、严格合并周期边界节点后得到。**这是直接送进 Abaqus 的模型。**

> 物理边长 $L$ 的确定：固定杆截面半径 $r=1$、目标相对密度 $\rho=0.1$，由单胞总杆长反推 $L$，使最终模型相对密度恰为 0.1。

## 三、仿真方法

仿真目标：对每个结构做单轴大变形压缩，得到应力–应变响应，用于材料筛选。采用 Abaqus/Explicit 显式动力学。

### 3.1 从杆系到有限元模型

- 取 `crystal_4x4x4` 的 $4\times4\times4$ 扩胞模型；
- 每根杆（`element_conn` 里的一条边）建成**圆截面梁**，截面半径 $r=1$，每根杆沿轴向划分为 3 段单元；
- 节点即梁单元的端点，杆与杆在公共节点处刚性连接。

### 3.2 加载与边界条件

- 模型上下各放一块刚性板；
- 上板沿 $z$ 轴单调压缩，直到名义应变 $\varepsilon=-0.30$（压缩 30%）；
- 板与结构、结构自身之间用 general contact，摩擦系数 $\mu=0.3$，防止穿透；
- 显式动力学准静态加载。

### 3.3 材料

主用 TPU95A，理想弹塑性（完美塑性），单位制 mm–N–MPa–tonne：

| 参数 | 值 |
|---|---|
| 杨氏模量 $E$ | 8.925 MPa |
| 泊松比 $\nu$ | 0.48 |
| 屈服应力 $\sigma_y$ | 3.207 MPa |

### 3.4 输出与指标

仿真直接输出上板的反力–位移，换算为名义**应力–应变曲线** $\sigma$–$\varepsilon$（每个结构一条，存为 csv）。在曲线上提取 9 个大变形压缩指标用于筛选：

| 指标 | 含义 |
|---|---|
| $E$ | 初始弹性模量（曲线起始斜率） |
| $\sigma_{\text{peak}}$ | 初始峰值应力 |
| $\sigma_{fp}$ | 首个平台/坍塌应力 |
| $W_v$ | 体积吸能（曲线下面积） |
| $\sigma_{pl}$ | 平台应力 |
| CFE | 压溃力效率 $=\sigma_{pl}/\sigma_{fp}$ |
| drop\_ratio | 峰后跌落比 |
| $\delta$ | 应力波动幅度 |
| $\varepsilon_{cd}$ | 致密化起始应变 |

每群约 25000 个结构各跑一条曲线，汇总成跨群榜单，再做极端性能筛选。

---

**链路一句话总结**：群矩阵 → 解对称约束得自由变量 → 在自由变量空间采样 + 选杆 → 拼单胞、扩胞成 `node_data`+`element_conn` → 梁单元压缩仿真 → $\sigma$–$\varepsilon$ → 9 指标筛选。
